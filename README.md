# 星幕塔罗

川川专用的中文塔罗占卜单页网站。

## 功能

- 每日一牌
- 三牌占卜
- 内置 22 张大阿尔卡那牌义
- 洗牌、翻牌、星幕、光晕和粒子交互
- 纯静态页面，无需后端、数据库或 API key

## 部署到 GitHub Pages

1. 在 GitHub 创建一个新的公开仓库，例如 `tarot-site`。
2. 把本地仓库推送到 GitHub：

```bash
git remote add origin https://github.com/YOUR_USERNAME/tarot-site.git
git push -u origin main
```

3. 打开仓库的 `Settings` -> `Pages`。
4. `Build and deployment` 选择 `Deploy from a branch`。
5. Branch 选择 `main`，目录选择 `/root`，保存。

上线后访问地址通常是：

```text
https://YOUR_USERNAME.github.io/tarot-site/
```
